# CEH -----Footprinting -- Recon-----



**Footprinting & Reconnaissance**
!!!!!! Techniques used during CEH learning !!!!!!
This repo accompanies a demonstration where i used open-source tools & online search engines / services to gather public information about target in a safe , legal lab environment.
*important* : All scans & recon shown in this project were performed only on systems i own or have explicit permission to test / practices labs .


##### What this demo shows ----

*Passive Information Gathering* 
***Tools***
//Whois lookups(Domain,Services,Creation date,Name servers).
//DNS Footprinting (DNS-Dumpster,nslookup,netcarft).
//Shodan(public asset discovery).

*Active Information Gathering*
***Tools***
//Nmap-zenmap(scanning open ports,services & version,Host discovery ,Network discovery)
//Theharvester
//Sherlock
